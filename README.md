
# Restaurant Location Recommendation System

This project predicts the best area to open a new restaurant using a simple machine learning model.
It trains a regression model and recommends the location with the highest predicted success score.

## Structure
- data/restaurant_data.csv
- src/recommender.py
- notebooks/model_training.ipynb
- requirements.txt

## Run
pip install -r requirements.txt
python src/recommender.py
